# GPS hexpansion

I came into posession of 95 surplous Ublox MAX-M10S GPS modules.
I just had to put some of them down on a hexpansion.


## Acknowledgements

Uses the [hexpansion template from EMF](https://github.com/emfcamp/badge-2024-hardware/tree/main/hexpansion)

I have attempted to maintain some pinout compatibility with [TechCabin's design](https://github.com/TechCabin/EMFBadge-Hexpansions-GPS), but it uses a very different chip.
Sorry for stepping on your toes!
